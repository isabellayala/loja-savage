# Loja de Roupas Responsiva

Este projeto faz parte da minha evolução como **Desenvolvedora Front-End**. Estou construindo uma loja de roupas utilizando **HTML5** e **CSS3**, seguindo a metodologia **Mobile First** para garantir uma experiência responsiva em diferentes dispositivos.

Atualmente estou desenvolvendo o **primeiro layout da loja**, enquanto continuo aprimorando meus conhecimentos e me preparando para expandir o projeto com **JavaScript** e **Python**.

---

## Tecnologias

- HTML5
- CSS3
- Google Fonts

---

##  Responsividade — Mobile First

**Celular** → estilo base

 **Tablet** → `@media (min-width: 768px)`

**Desktop** → `@media (min-width: 1024px)`
